### 场景1: 计算某职工月工资

**操作步骤:**
1. 进入"工资管理" → "工资计算"
2. 选择职工: "张三 (EMP001)"
3. 选择月份: "2024-01"
4. 点击"计算工资"按钮

**系统自动执行:**
```
1. 读取职工信息
   - 岗位等级: 专技
   - 职称等级: 专技十级
   - 薪级: 25
   - 学历: 硕士研究生

2. 查询岗位工资标准
   → 专技十级 = 6,500元

3. 查询薪级工资标准
   → 25级 = 2,250元

4. 计算绩效工资
   → (6,500 + 2,250) × 0.4 = 3,500元

5. 添加津贴补贴
   → 住房补贴 800元
   → 交通补贴 500元

6. 计算社保扣款
   → 养老保险: 8,750 × 8% = 700元
   → 医疗保险: 8,750 × 2% = 175元
   → 失业保险: 8,750 × 0.5% = 43.75元
   → 住房公积金: 8,750 × 12% = 1,050元

7. 计算个人所得税
   → 应纳税所得额 = 13,550 - 2,268.75 - 5,000 = 6,281.25元
   → 适用税率: 10%, 速算扣除数: 2,520
   → 个税 = 6,281.25 × 10% - 2,520 = 285.50元

8. 汇总输出
   → 应发: 13,550元
   → 扣款: 2,254.25元
   → 实发: 11,295.75元
```

**结果展示:**
- 所有明细项以表格形式展示
- 应发项目绿色显示
- 扣款项目红色显示
- 实发工资蓝色加粗放大
