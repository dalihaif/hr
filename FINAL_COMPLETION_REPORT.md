## 📞 技术支持

### 常见问题

**Q1: 如何配置职工的岗位等级和薪级?**
```
职工管理 → 编辑职工 → 事业单位人员信息
- 岗位等级: 选择管理/专技/工勤
- 职称等级: 输入如"专技十级"
- 薪级: 输入1-65的数字
```

**Q2: 工资计算结果为0怎么办?**
```
检查项:
1. 职工是否配置了岗位等级
2. 职工是否配置了薪级
3. 岗位工资标准表是否有数据
4. 薪级工资标准表是否有数据
```

**Q3: 如何添加自定义工资项目?**
```
工资管理 → 工资项目管理 → 新增项目
- 填写项目名称和编码
- 选择计算方式(固定值/比例/公式)
- 设置是否计税
```

**Q4: 社保比例如何调整?**
```
工资标准配置 → 社保缴费配置 → 编辑配置
- 修改各险种比例
- 调整缴费基数范围
- 保存生效
```
