#### 核心业务逻辑

**1. 绩效金额计算**
```python
# 基于科室岗位系数和考核评分
performance_amount = base_performance × coefficient × (total_score / 100)

# 示例:
# 内科医师,系数1.2,基准3000元,评分92分
# 绩效金额 = 3000 × 1.2 × 0.92 = 3312元
```

**2. 总分计算**
```python
# 加权平均法
total_score = Σ(单项得分 × 权重 / 满分)

# 等级判定
if total_score >= 90: grade = '优秀'
elif total_score >= 75: grade = '良好'
elif total_score >= 60: grade = '合格'
else: grade = '不合格'
```

**3. 产假天数计算**
```python
# 医院专用规则
base_days = 158  # 基础产假(国家标准98天 + 云南奖励60天)
extra_days = 0

if birth_type in ['难产', '剖腹产']:
    extra_days += 15

if is_multiple_birth and multiple_count > 1:
    extra_days += (multiple_count - 1) × 15

total_days = base_days + extra_days
```
